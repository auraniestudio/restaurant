# Your images folder

The app now uses **your own photos** from this folder (no external links).

## Add these files

### Gallery (6 photos)
Put 6 images in **`gallery/`** with these names:
- `gallery/1.jpg`
- `gallery/2.jpg`
- `gallery/3.jpg`
- `gallery/4.jpg`
- `gallery/5.jpg`
- `gallery/6.jpg`

### Featured dishes (4 photos)
Put 4 images in **`dishes/`** with these names:
- `dishes/tomato-salad.jpg` — Heirloom Tomato Salad
- `dishes/truffle-risotto.jpg` — Truffle Risotto
- `dishes/wagyu-ribeye.jpg` — Grilled Wagyu Ribeye
- `dishes/chocolate-souffle.jpg` — Chocolate Soufflé

### Optional: social preview
- **`../og-image.jpg`** (in `public/`) — image for Facebook/Twitter link previews (~1200×630 px).

You can use `.jpg`, `.jpeg`, or `.png`; if you use different extensions, update the paths in `src/components/Gallery.tsx` and `src/data/dishes.ts`.
